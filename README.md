# 项目简介

该项目是一款基于Spring Boot和Vue.js协同过滤的电影推荐系统。该系统旨在为用户提供个性化的电影推荐服务，并具备完善的管理功能，以支持电影内容的运营和用户管理。

## 核心功能

- 用户端：
  - 首页：展示最新电影资讯和推荐电影
  - 付费电影：根据用户订单历史推荐电影
  - 电影论坛：用户交流电影心得和讨论
  - 电影资讯：提供电影新闻和行业动态
  - 免费电影：根据用户收藏推荐电影
  - 收藏电影：用户保存感兴趣的电影
  - 购买订单：展示用户购买记录
  - 评论：用户对电影发表评论
  - 播放视频：在线观看电影
  - 个人中心：管理个人资料和设置

- 管理端：
  - 用户管理：管理用户账户和权限
  - 电影管理：上传、分类和更新电影信息
  - 评论管理：监控和审核用户评论
  - 资讯管理：发布和管理电影资讯
  - 论坛管理：维护论坛秩序和内容
  - 收藏管理：管理用户收藏的电影
  - 轮播图设置：定制首页轮播图内容
  - 基础数据管理：管理分类等基础数据

## 适用场景

该系统适用于在线电影平台、电影社区以及个人或企业建立的电影推荐服务。

## 技术特点

- 采用Spring Boot和Vue.js技术栈，实现前后端分离，提高开发效率和用户体验。
- 基于协同过滤算法提供个性化的电影推荐，提高推荐效果。
- 系统结构清晰，模块化设计，便于扩展和维护。
- 支持多种浏览器和移动设备访问，确保系统的高可用性和可访问性。
